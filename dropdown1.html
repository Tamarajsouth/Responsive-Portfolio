Over Yonder

The avid outdoor lover's new favorite app. Determines user location and populates hikes nearby using REI API. A dynamic user interface utilizing Materialize CSS and jQuery. OpenWeatherMaps API also used to show current weather conditions for the proposed hiking location. A collaborative group project, I was responsible for creating the user interface functionality and design of the "home" and "about us" page.

My Familiar

My Familiar is an RPG character generator that allows the user to create player characters and NPCs that are compatible with most d20 RPGs. This web application was developed with the purpose of transporting the currently quarantined masses to far off lands and imagining themselves interacting with all sorts of creatures and characters. My Familiar uses the Bootstrap framework to provide a professional and fantastic feel, while implementing the parallax feature for and more elevated aesthetic. My Familiar is a full stack web application deployed to Heroku. Another collaborative project, I was tasked with organzing data from the data base into forms to create the character cards.

Technologies Used:

  • Node and Express Web Server
  • jQuery
  • MySQL - Sequelize ORM
  • Heroku (with Data)
  • Socket.io
  • Bootstrap 4
  • Google API
  • inDesign
  • ESLint

    Eat-Da-Burger

    a burger logger with MySQL, Node, Express, Handlebars and a homemade ORM (yum!) Eat-Da-Burger is a restaurant app that lets users input the names of burgers they'd like to eat. Whenever a user submits a burger's name, the app will display the burger -- waiting to be devoured. Each burger in the waiting area also has a Devour it! button. When the user clicks it, the burger will move to the bottom of the page under the Devoured menu. The app stores every burger in a database, whether devoured or not.

    Technologies Used:

  • Node.js
  • Express
  • Express Handlebars npm package
  • MySQL npm package and workbench
  • JawsDB extension
  • Bootstrap 4
  • Heroku
